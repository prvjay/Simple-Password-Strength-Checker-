Overview

The Password Strength Checker is a simple Python program that helps users evaluate the security of their passwords. 
It checks for common security criteria such as length, uppercase and lowercase letters, numbers, and special characters. 
This project demonstrates fundamental programming skills and basic cybersecurity awareness.

Features:
 1.Interactive password input
 2.Checks for:
  Minimum length (8 characters)
  Uppercase letters
  Lowercase letters
  Numbers
  Special characters
  Provides immediate feedback on password strength
  Easy to read and beginner-friendly Python code

Installation
 1.Make sure you have Python 3.x installed on your system.
 2.No additional libraries are required.

Usage
 1.Open a terminal or your Python IDE (like PyCharm).
 2.Run the program
 3.Enter your password when prompted.
 4.The program will display feedback for each security criteria.


Example Output:

Contains lowercase letters.
Contains numbers.
Contains special characters.
